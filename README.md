# PowerBI-Music-metrics-analysis
Analysis about Music Data: To study how specific traditional melodies (Ragas) and track features behave in a database.

The core of this project is a high-performance, interactive Power BI dashboard designed to make complex musical and psychological data immediately understandable.
File-Based Data Modeling: Imported, cleaned, and structured the core track metadata, traditional raga_label categories, and Mental_Health_Labels entirely within the Power BI environment.

Custom Dark-Themed UI: Developed using a unified visual theme—featuring a premium Luxury Gold & Onyx color palette optimized for dark canvas backgrounds.
Advanced Interactive Slicers: Implemented global filters allowing users to toggle seamlessly between different mental health profiles (Anxiety, Depression, Normal) and observe real-time changes in audio metrics.

Multi-Page Report Architecture: Structured into distinct analytical sheets, including a dedicated section (Sheet 3) for mapping behavioral profiles directly against traditional Raga distributions and complex audio features.

Cross-Visual Filtering: Configured precise interactive behaviors between multi-series column charts, line distributions, and single-metric KPIs to ensure a smooth, intuitive user experience.
